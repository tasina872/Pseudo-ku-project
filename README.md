# Pseudo-ku-project
(by: Tasin Abdullah)

// DESCRIPTION + SUMMARY \\
This program generates a random, solved Sudoku board by first generating the values of the first row of numbers, starting at a random number and increasing by 1 (if this number is greater than 9, it's automatically set to 1 to comply with Sudoku rules). Then, each subsequent row is generated the same way, but each number is shifted by a value of 3. For every multiple of 3rd row, the subsequent multiple of 4th row is shifted by 4.
Check the design document for a more detailed explanation.
